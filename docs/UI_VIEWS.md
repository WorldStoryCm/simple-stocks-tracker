# Stock Tracking App — UI Views

## 1. Dashboard
Main purpose: answer “How am I doing?”

Sections:
- portfolio summary cards
- realized P/L cards (today, week, month, all-time)
- weekly goal progress card
- platform summary table
- bucket budget usage
- recent trades
- top gainers / losers by realized P/L

---

## 2. Trades page
Main purpose: raw ledger management.

Layout:
- top filters row
- trades table
- side drawer or modal for add/edit trade
- row action menu

Filters:
- date range
- platform
- symbol
- bucket
- action type

---

## 3. Positions page
Main purpose: current exposure.

Layout:
- filters
- grouped table by platform
- expandable symbol rows if needed

---

## 4. Performance page
Main purpose: period-based profit/loss analysis.

Tabs:
- Daily
- Weekly
- Monthly
- Yearly

Possible chart:
- cumulative realized P/L over time

---

## 5. Platforms page
Main purpose: manage broker accounts and see broker-level totals.

---

## 6. Watchlist page
Main purpose: pre-trade ideas and monitoring.

Layout:
- watchlist table or cards
- add/edit item dialog
- status filter
